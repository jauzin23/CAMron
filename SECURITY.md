# Security Policy

We take security seriously. If you find a security vulnerability in this project, please do not share it publicly. Reporting security issues in public GitHub issues can expose users to risk before a fix is available.

## Reporting a Vulnerability

To report a security vulnerability, send an email to jaam.joao.monteiro.2008@gmail.com.

In your email, please describe the vulnerability clearly. Include the steps needed to reproduce the issue, the potential impact, and any other relevant information.

We will review your report as soon as possible and work on a fix. We ask that you give us time to investigate and resolve the issue before you disclose it publicly.
